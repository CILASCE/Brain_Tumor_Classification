# 🧠 Brain Tumor MRI Scans Dataset
### 📖 Description
This dataset contains Brain Magnetic Resonance Imaging (MRI) scans, classified into four distinct categories:
* **Glioma**
* **Meningioma**
* **Pituitary Tumor**
* **No Tumor** (Normal)

### 📥 Where to Find the Data
You can download the complete dataset directly from Kaggle using the link below:

🔗 **[Brain Tumor MRI Scans on Kaggle](https://www.kaggle.com/datasets/rm1000/brain-tumor-mri-scans)**
